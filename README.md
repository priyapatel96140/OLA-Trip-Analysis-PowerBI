# OLA Trip Analysis Dashboard

## Brief Summary
Power BI dashboard project analyzing OLA trip performance, driver earnings, and ride trends.

## Overview
This project provides insights into OLA ride bookings, trip duration, fare distribution, and driver earnings using an interactive Power BI dashboard.

## Problem Statement
The goal of this project is to analyze ride activity and identify trends in trips, searches, fares, and driver performance to support business decision-making.

## Dataset
The dataset contains:
- Trip details
- Fare information
- Search statistics
- Driver earnings
- Distance and duration metrics
- Area-wise trip records

## Tools and Technologies
- Power BI
- Excel
- Data Visualization

## Dashboard Features
- Total completed trips
- Total searches and estimates
- Fare vs duration analysis
- Distance vs duration trends
- Driver earnings KPI
- Area-wise trip distribution map
- Interactive filters and slicers

## Key Insights
- Total completed trips reached 903
- Driver earnings exceeded 696K
- Search count was significantly higher than completed trips
- Certain assemblies showed higher trip activity
- Trip duration directly impacted fare and distance

## Output
Interactive Power BI dashboard with KPIs, trend analysis, and geographic trip visualization.

## How to Run this Project
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh dataset if required

## Results & Conclusion
The dashboard helps analyze operational performance, customer demand, and driver earnings trends for better business insights.
